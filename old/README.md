# Stats Salad Dashboard

A React dashboard website using Material-UI, Chart.js, and initial structure for globe.jl integration.

## Structure
- `src/` - React source code
- `public/` - Static assets
- `scripts/` - Placeholder for Julia (globe.jl) integration
- `docs/` - Documentation

## Getting Started
1. Install dependencies: `npm install`
2. Start development server: `npm start`

## Tech Stack
- React
- Material-UI
- Chart.js
- Julia (globe.jl, planned)
